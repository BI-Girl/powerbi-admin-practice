# Workspace Governance Standards
## Naming Convention
- Pattern : [Domain]_[Project]_[Environment]
- Example : FMCG_SupplyChain_Prod
## Access Roles
- Admin : BI Platform Team Only
- Member: Senior BI Developers
- Contributor : Bi Developers and Data Engineers
- Viewer : Business Stakeholders
## Review Cadence
- Qurterly access review for all workspaces
- Inactive workspaces flagged after 60 days
